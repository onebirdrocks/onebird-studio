## 🚀 Features

- 🧠 Built with **Electron**
- ✨ Written in **TypeScript**
- ⚙️ Bundled with **Webpack**
- ⚡ Hot-reloading for fast development
- 🖥️ Cross-platform (macOS, Windows, Linux)

```bash
# Clone the repo
